

# PSP - Programación de Servicios y Procesos

Repositorio dedicado a las prácticas y ejercicios del módulo **PSP (Programación de Servicios y Procesos)**.

##  Contenidos
El repositorio incluye ejemplos y tareas sobre:
- **Programación Multihilo:** Gestión de hilos (Threads), sincronización y comunicación entre procesos.
- **Comunicaciones en Red:** Creación de sockets TCP/UDP, arquitectura cliente-servidor.
- **Servicios en Red:** Protocolos estándar (HTTP, FTP, SMTP).
- **Seguridad:** Cifrado de información y firmas digitales.

##  Proyectos destacados
*Aquí puedes añadir nombres de carpetas específicas si las tienes, por ejemplo:*
- `Práctica 1: Sockets`: Implementación de un chat básico.
- `Práctica 2: Hilos`: Simulación de un sistema de reservas concurrente.

## 🛠️ Requisitos
- Java JDK 17 o superior.
- Conocimientos básicos de redes y concurrencia.
